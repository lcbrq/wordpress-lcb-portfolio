Displays portfolio including custom image with short description.

== Description ==
This plugin adds custom post type portfolio to your Wordpress. Plugin allows to add image and description of your work including a normal or external post link url (permalink). 
By default, you can setup your portfolio output in one, two, three or four columns design. Polish language support included.

== Installation ==

From your WordPress dashboard

   1. Visit 'Plugins > Add New'
   2. Search for 'LCB portfolio'
   3. Activate LCB portfolio from your Plugins page.

From WordPress.org

   1. Download LCB portfolio.
   2. Upload the 'lcb-portfolio' directory to your '/wp-content/plugins/' directory
   3. Activate LCB portfolio from your Plugins page.

You will find 'Portfolio' menu in your WordPress admin panel. To display portfolios you should use shortcode [portfolio] on Your site.
